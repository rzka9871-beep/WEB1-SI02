FAQ

1. Apa itu JavaScript?
JavaScript adalah bahasa pemrograman untuk web.
2. Apa itu jQuery?
jQuery adalah library JavaScript.
3. Apa fungsi DOM?
DOM digunakan untuk memanipulasi elemen HTML.
4. Apa itu selector?
Selector digunakan untuk memilih elemen.
5. Apa itu event?
Event adalah aksi pengguna.
